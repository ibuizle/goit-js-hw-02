# JavaScript Fundamentals: Arrays & Functions 🔢

This repository contains the second homework assignment for the GoIT Fullstack Developer course. It focuses on the core building blocks of JavaScript logic: working with **arrays**, creating **reusable functions**, and implementing **loops**.

🔗 **[Live Demo](https://ibuizle.github.io/goit-js-hw-02/)**

## 📖 About the Project

The goal of this assignment was to practice structured programming. The tasks involve solving logic problems by processing lists of data, manipulating strings, and wrapping code into functions to make it modular and clean.

### Key Learnings & Features:
- **Array Operations:** Adding, removing, and finding elements using methods like `push()`, `slice()`, `split()`, and `join()`.
- **Functional Programming:** Writing functions with parameters and return values to solve specific tasks.
- **Loops & Iteration:** Using `for` and `for...of` loops to traverse arrays and process data.
- **String Manipulation:** Transforming text data (e.g., creating slugs from titles or shortening descriptions).
- **Code Logic:** Implementing conditional statements (`if...else`) within functions to handle different scenarios.



## 🛠 Built With

- **HTML5:** Basic structure for running scripts.
- **Vanilla JavaScript:** ES6+ core features focusing on arrays and functions.
